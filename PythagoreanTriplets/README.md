# Pythagorean Triplets

---

## Problem Statement

Input three integers `n1` , `n2` , `n3` . <br>Print `true` if they form a Pythagorean triplet and `false` if they do not.

## Constraints
2 <- `n1` <- 10^9 <br>
2 <- `n2` <- 10^9 <br>
2 <- `n3` <- 10^9

---

### Example
Input: <br>
`n1`<br>`n2` <br>`n3`
<br><br>
Output:<br>
`true` / `false`
