# Print fibonacci series

## Problem Statement

Input an integer `n`. <br>
Print first `n` numbers of the fibonacci series.

---

## Constraints

1 <- `n` < 40

---

## Input format

`n`

---

## Output format 

0 <br>
1 <br>
1 <br>
2 <br>
3 <br>
... first `n` fibonaccis

---

## Example

### Input

10

### Output

0 <br>
1 <br>
1 <br>
2 <br>
3 <br>
5 <br>
8 <br>
13 <br>
21 <br>
34 <br>
