# Benjamin Bulbs

## Problem Statement
You are given `n` number of bulbs switched off initially. A weird voltage fluctuation hits the circuit `n` times. In the 1st fluctuation, all bulbs are toggled, in the 2nd fluctuation, every 2nd bulb is toggled, in the 3rd fluctuation, every 3rd bulb is toggled. Similarly `n` fluctuations take place. You have to find which bulbs will be switched on after all fluctuations. 

---

## Input Format
`n`
<br>
representing the number of bulbs.

---

## Output Format
`b1`<br>
`b2`<br>
`b3`<br>
.. all switched on bulbs.

---

## Constraint
2 <- `n` <- 10^9
