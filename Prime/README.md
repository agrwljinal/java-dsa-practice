# Prime Number Queries

## Problem Statement

You input an integer `t`, representing the number of test cases.  
For each test case, you input an integer `n`.  

For every `n`, determine whether it is a prime number or not.

---

## Constraints

- 1 < `t` < 10000  
- 2 < `n` < 10^9  

---

## Input format

- First line: integer `t` (number of test cases)  
- Next `t` lines: each line contains an integer `n` 

---

## Output format

- First line: integer `t` (number of test cases)  
- Next `t` lines: each line contains an integer `n` 

---

## Example

### Input

3
7
8
13

### Output

prime 
not prime
prime
