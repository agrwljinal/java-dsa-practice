# Element in an Array

## Problem Statement
Given a number `n` representing size of an array. Given `n` elements of the array.
<br>Another number `d` is given. Find the index of `d` and -1 if `d` is not in array.

--- 

## Input

`n`<br>
`n` elements
<br>`d`

--- 

## Output

Index of `d`

---

### Example

Input:<br>
3<br>10<br>20<br>30<br>20
<br><br>
Output:<br>
1
