# Span of an array

---

## Problem Statement
Given size of array `n`.
<br>
`n` numbers of array are also given.

--- 

## Input
`n`
<br>
`n` numbers

---

## Output
`max` - `min`

---

## Constraints
1 <- `n` <- 10^4 <br>
0 <- elements <- 10^9

### Example

Input: <br>
5<br>10<br>20<br>30<br>40<br>50
<br><br>
Output:<br>
40
