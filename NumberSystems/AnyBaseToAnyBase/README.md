# Any Base to Any Base

---

## Problem Statement
You are given a number `n`on base `b1`. <br>Convert this number into its equivalent number in base b2.

---

## Input Format
`n`
<br>
`b1`
<br>
`b2`

---

## Output Format
Number equivalent to `n` in base `b2`.

---

## Constraints
2 <- `b1` <- 10
<br>
2 <- `b2` <- 10

---

### Example
Input: <br>
111001 <br>
2 <br>
10
<br><br>
Output: <br>
57
