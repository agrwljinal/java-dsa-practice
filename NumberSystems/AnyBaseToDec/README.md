# Any Base to Decimal

---

## Problem Statement
You are given a number `n`on base `b`. <br>Convert this number into its equivalent number in base 10.

---

## Input Format
`n`
<br>
`b`

---

## Output Format
Number equivalent to `n` in base 10.

---

## Constraints
2 <- `b` <- 10

---

### Example
Input: <br>
111001 <br>
2
<br><br>
Output: <br>
57
