# Any Base Subtraction

---

## Problem Statement
You are given a base `b` and two numbers `n1` and `n2` in this base. <br>Subtract `n1` - `n2` in base `b`.

---

## Input Format
`b`
<br>
`n1` <br>
`n2`

---

## Output Format
Difference of `n1` and `n2` in base `b`.

---

## Constraints
2 <- `b` <- 10 <br>
`n1` <- `n2`

---

### Example
Input: <br>
8 <br>
1212 <br>
754

<br><br>

Output: <br>
236
