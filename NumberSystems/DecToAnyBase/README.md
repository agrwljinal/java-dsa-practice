# Decimal to Any Base

---

## Problem Statement
You are given a decimal number `n`. <br>Convert this number into its equivalent number in base `b`.

---

## Input Format
`n`
<br>
`b`

---

## Output Format
Number equivalent to `n` in base `b`.

---

## Constraints
2 <- `b` <- 10

---

### Example
Input: <br>
21 <br>
2

<br><br>

Output: <br>
10101
