# Any Base Addition

---

## Problem Statement
You are given a base `b` and two numbers `n1` and `n2` in this base. <br>Add `n1` and `n2` in base `b`.

---

## Input Format
`b`
<br>
`n1` <br>
`n2`

---

## Output Format
Sum of `n1` and `n2` in base `b`.

---

## Constraints
2 <- `b` <- 10 

---

### Example
Input: <br>
8 <br>
734 <br>
256

<br><br>

Output: <br>
1212
