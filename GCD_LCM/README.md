# Print GCD (Greatest Common Divisor) and LCM (Least Common Multiple) of two numbers

## Problem Statement
Input two integers `num1` and `num2`. Calculate and print their GCD and LCM.

### Input Format
`num1`<br>
`num2`

### Output Format
`GCD`<br>
`LCM`

### Contraints
2 <- `num1` <- 10^9 <br>
2 <- `num2` <- 10^9

### Example
Input:<br>
24<br>
36
<br> <br>
Output:<br>
12<br>
72
